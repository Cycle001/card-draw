# card-draw - 抽卡模拟器

## 项目介绍

这是一个功能强大的卡牌抽取模拟器，专为斗地主、升级等棋牌游戏设计。程序采用直观的图形界面，支持完整的卡牌管理、随机抽取和状态记录功能。

## 主要功能

### 1. 模拟抽牌
- **随机抽牌**: 从完整卡牌库中随机抽取一张未被抽取过的卡牌
- **手动标记**: 点击左侧卡牌区域可以手动标记卡牌为"已抽取"状态
- **实时显示**: 右侧区域显示当前抽取到的卡牌

### 2. 智能记牌
- **视觉标记**: 已抽取的卡牌会以浅灰色显示，清晰区分已出和未出的牌
- **状态记忆**: 程序会记住所有卡牌的抽取状态，方便统计

### 3. 卡牌管理
- **默认卡组**: 内置完整扑克牌组（54张）和两副扑克牌组（108张）
- **自定义卡组**: 支持创建、编辑和删除自定义卡组
- **文件操作**: 支持从文件导入卡组和导出卡组到文件

## 运行方式

### 方法一：直接运行Python脚本
```bash
python Card_draw_simulator.py
```

### 方法二：打包为可执行文件
```bash
pyinstaller --onefile --noconsole Card_draw_simulator.py
```

## 界面说明

### 主界面布局
程序界面分为三个主要区域：

#### 1. 顶部工具栏
- **卡组选择**: 下拉菜单选择不同卡组
- **增加卡组**: 创建新的自定义卡组
- **删除卡组**: 删除当前选中的自定义卡组
- **修改卡组**: 进入卡组编辑模式
- **保存卡组**: 保存编辑后的卡组
- **载入卡组**: 从文本文件导入卡组
- **导出卡组**: 将当前卡组导出到文件
- **状态显示**: 显示当前卡组名称和牌数

#### 2. 左侧卡牌显示区
- 显示当前卡组的所有卡牌
- 点击卡牌可切换"已抽取"状态
- 已抽取的卡牌显示为浅灰色
- 支持自适应布局，根据窗口大小自动调整每行显示数量

#### 3. 右侧抽牌区
- **当前抽到的牌**: 显示最新抽取的卡牌
- **随机抽一张牌**: 随机抽取一张未抽过的牌
- **重置按钮**: 清空所有抽取记录，恢复初始状态

## 卡牌排序规则

程序按照特定的顺序排列卡牌：
1. **花色顺序**: 方块 > 梅花 > 红心 > 黑桃
2. **点数顺序**: K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3 > 2 > A
3. **特殊牌**: 大小王位于最后（先小王后大王）

## 编辑模式

### 进入编辑模式
点击"修改卡组"按钮进入卡组编辑模式

### 编辑功能
1. **多选操作**: 左键点击选择多张卡牌
2. **右键菜单**: 右键单张卡牌弹出菜单
   - 重命名卡牌
   - 删除单张卡牌
3. **批量操作**:
   - 删除选中卡牌：删除所有选中的卡牌
   - 添加新卡牌：点击"+"按钮添加新卡牌
   - 恢复默认：恢复默认卡组的原始状态

### 退出编辑模式
点击"保存卡组"按钮保存修改并退出编辑模式

## 文件格式

### 导入/导出文件格式
程序支持纯文本格式（.txt）和CSV格式（.csv）：
- 每行一张卡牌名称
- 示例：
  ```
  K♦
  Q♦
  J♦
  10♦
  大王
  小王
  ```

## 使用场景

1. **棋牌游戏训练**: 帮助玩家练习记牌技巧
2. **概率统计**: 模拟卡牌抽取过程，统计概率

## 注意事项

1. 程序默认提供了两种标准卡组，可以直接使用
2. 自定义卡组支持任意卡牌名称，包括特殊符号
3. 编辑模式下，卡牌的颜色标记会暂时失效
4. 程序会保存最近使用的卡组状态，重启后需要重新设置
